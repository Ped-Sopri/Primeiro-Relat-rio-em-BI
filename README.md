## 📝 Descrição do Projeto
Este repositório contém o meu primeiro projeto de Business Intelligence desenvolvido no **Power BI**. O objetivo principal foi transformar dados brutos de vendas em informações estratégicas e visuais, permitindo uma análise clara do desempenho comercial.

O dashboard foi construído para responder a perguntas de negócio fundamentais, como tendências de faturação, desempenho por categoria e análise geográfica.

## 🚀 Funcionalidades e Análises
* **Visão Geral de Vendas:** Monitorização da receita total, margem de lucro e quantidade vendida.
* **Análise Temporal:** Comparação de vendas ao longo dos meses para identificar sazonalidade.
* **Segmentação de Produtos:** Identificação dos produtos e categorias mais vendidos.
* **KPIs de Performance:** Indicadores chave para apoio à tomada de decisão rápida.

## 🛠️ Ferramentas Utilizadas
* **Power BI Desktop:** Para extração, modelagem e visualização de dados.
* **Power Query (ETL):** Limpeza e transformação de dados.
* **Linguagem DAX:** Criação de medidas calculadas personalizadas.
* **Excel/CSV:** Como fonte primária de dados.

## 📈 Modelagem de Dados
O projeto seguiu as melhores práticas de BI, incluindo:
1.  **ETL:** Tratamento de valores nulos e formatação de colunas.
2.  **Star Schema:** Criação de tabelas de Dimensão e Fato para otimizar a performance do relatório.
3.  **Cálculos DAX:** Implementação de medidas como Total de Vendas, Margem % e Acumulados.
